注意事項

📅 第一天
✓
確認烤肉食材採購由誰負責
✓
確認素食人數，準備素食選項
✓
司機與車輛分配、停車費分攤
✓
湖心庭民宿房型與床位確認
✓
準備現金（部分景點僅收現金）
✓
下載離線地圖（山區訊號較弱）
✓
長輩、小孩用藥與特殊需求確認
✓
雨天備案景點（日月潭文武廟等）
📅 第二天
✓
埔里鹹油條：前一晚先用 LINE 預訂好，確認份數
✓
阿豐師餐廳：事先確認是否需要訂位（約 NT$450/人）
✓
妮娜巧克力 DIY：確認是否需要提前預約活動
✓
Jay's Pizza 外送：確認民宿 9 折優惠使用方式、提早下單
✓
確認 Day 2 素食人數與 Jay's Pizza 素食口味選擇
✓
晚會活動安排（桌遊、照片分享等）
